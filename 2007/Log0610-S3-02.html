Ruby-GNOME2とは

GNOMEの提供するCのライブラリをRubyからアクセスできるようにしたもの

GNOMEのライブラリとは

  • 共通のライブラリなのでGNOMEがなくてもいい
    • GNOME嫌いなひともどうぞ
  • GUIツールキット、画像処理、その他
  • マルチプラットフォーム
    • Linux,FreeBSD
    • Mac OS X
    • MS Windows(Win32 Installerでバイナリ配布中）

Ruby-GNOME2のライブラリ

RubyらしいAPIを提供
block,引数の省略

Base

ほかのライブラリから呼び出される

Graphics

さまざまなフォーマットの画像文字など画像処理ライブラリ

GUI以外の用途でも使える

⇒Webアプリでもどうぞ

  • rcairo
    • 超協力描画エンジン
  • GdkPixbuf
    • 画像の読込・保存
  • RSVG, Poppler
    • SVG,PDFの読込

Widgets

GUI部品

  • GTK
    • 主要なWidget郡
  • GtkSourceView
    • 強力なテキストウィジェット
  • GnomeCanvas
    • 線、矩形、楕円などの図形を扱える

ちなみに今日はあえてWindowsでやってます

  • GtkGLExt
    • 3Dグラフィックス（OpenGL）
  • GtkMozEmbed
    • MozillaベースのHTML Widget
  • その他いろいろ

Misc

  • Glade, Libglade
    • GUIビルダを使ったRapidアプリ開発
  • GConf
    • アプリの設定値を保存・読込
      • レジストリみたいなもの
    • 値の変更をイベント通知できる
  • Ruby-GetText-Package
    • メッセージのローカライズ
    • Gladeとの相性もGood
