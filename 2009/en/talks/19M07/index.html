Title: iPhone Application Development with Ruby

Speaker

Takuma Mori (UICoderz)

Abstract

You can use MacRuby or RubyCooca to development MacOSX applications, however there is no Ruby implementation for iPhone application development so far. RubyCocoa for iPhone project I started, is now running on the iPhone devices and you can write some tiny applications. I'll introduce this RubyCocoa for iPhone project and pragmatic iPhone application development techniques using it.

Date

July 19, 2009

Room

Conference Room(2nd Floor)

Speaker Profile

I'm working on the development of iPhone Applications recently. A freelance programmer.
